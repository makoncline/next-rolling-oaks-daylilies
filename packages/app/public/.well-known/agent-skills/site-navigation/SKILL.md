# Rolling Oaks Daylilies Site Navigation

Use this skill when a user wants help finding their way around Rolling Oaks
Daylilies.

## Main Paths

- Home: `https://rollingoaksdaylilies.com/`
- Catalogs: `https://rollingoaksdaylilies.com/catalogs`
- Search: `https://rollingoaksdaylilies.com/catalog/search`
- For sale: `https://rollingoaksdaylilies.com/catalog/for-sale`
- All listings: `https://rollingoaksdaylilies.com/catalog/all`
- Cart: `https://rollingoaksdaylilies.com/cart`
- Blog: `https://rollingoaksdaylilies.com/blog`

## Guidance

- Send users to Search when they know a cultivar name, hybridizer, color,
  bloom traits, or price range.
- Send users to For Sale when they want purchasable plants.
- Send users to Catalogs when they want broad browsing by list.
- Use the cart as an inquiry aid, not as a guaranteed checkout flow.
- Contact forms and cart inquiries should be reviewed by the user before
  submission.
